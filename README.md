# 📦 光子盘点 —— 离线网页盘点工具

> 一款无需安装、无需联网即可使用的手机端盘点工具  
> 拍照 + 录入 + 本地存储 + Excel / ZIP 导出

---

## ✅ 功能特点

- 📸 **相机拍照**  
  调用手机摄像头，支持旋转、拖拽取景框

- 🧾 **商品信息录入**  
  产品名称 / 类别 / 规格 / 颜色 / 数量 / 备注

- 💾 **本地存储**  
  所有数据保存在浏览器 `localStorage` 中，刷新不丢失

- 📥 **备份 & 恢复**  
  支持创建本地备份、锁定重要备份、一键恢复

- 📤 **Excel + 图片打包导出**  
  自动生成：
  - `盘点数据.xlsx`
  - `images/` 文件夹（JPEG 图片）

- 📱 **移动端优化**  
  适配手机屏幕，按钮大、操作直观

---

## 🚀 使用方法

### 1️⃣ 打开工具
- 将 `盘点3.7性能优化版.html`  
  用 **手机浏览器** 打开（推荐 Chrome / Edge）

### 2️⃣ 开始盘点
1. 对准商品，点击 📸 拍照  
2. 填写商品信息  
3. 点击 ✅ 保存

### 3️⃣ 查看清单
- 点击 **📋 清单** 查看当前盘点记录
- 支持删除单条记录

### 4️⃣ 导出数据
- 点击 **📤 导出**
- 自动下载 `光子盘点_YYYYMMDD.zip`

解压后得到：
光子盘点_20260121.zip

├─ 光子盘点_20260121.xlsx

└─ images

├─ IMG_0001.jpg

├─ IMG_0002.jpg

└─ ...

---

## 🖼️ 图片质量控制

- 拍照保存：**65% JPEG**
- 导出 ZIP：**65% JPEG**
- 最大宽度：**800px**

👉 平衡清晰度与文件体积，适合 Excel 附件与微信传输

---

## 💡 注意事项

- ✅ **首次使用需允许摄像头权限**
- ✅ 数据仅保存在本机浏览器
- ⚠️ 清除浏览器缓存会清空数据
- ✅ 建议定期使用 **📥 备份**
- ✅ 导出前可先点击 **📋 清单** 确认数据
