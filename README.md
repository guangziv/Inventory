光子盘点 · Inventory Cam

一款运行在浏览器里的离线盘点工具，拍一拍、填一填，即可完成库存采集与导出。

✨ 功能特性

- 📷 相机盘点
   - 调用手机后置摄像头
   - 支持 自由拖拽 / 双指缩放 / 旋转 / 对焦框
   - 所见即所得，操作流畅
- 📝 商品录入
   - 产品名称 / 类别 / 规格 / 颜色 / 数量 / 备注
   - 自动保存拍摄图片
   - 表单与相机画面无缝衔接
- 📋 清单管理
   - 实时清单列表
   - 支持 单条记录删除
   - 表格自适应屏幕，操作列常驻右侧
- 📦 数据导出
   - 一键导出 Excel + 图片
   - ZIP 包自动按日期命名
   - 图片与 Excel 一一对应，便于核对
- 💾 本地备份 & 恢复
   - 支持本地历史备份（最多 5 份）
   - 误删可恢复，数据安全有保障
- 🔒 完全离线
   - 所有数据保存在浏览器本地
   - 无需联网、无需登录、无需服务器

🚀 快速开始

在线使用（推荐）

👉 https://guangziv.github.io/Inventory/

本地使用

git clone https://github.com/guangziv/Inventory.git
cd Inventory
# 直接用浏览器打开 index.html 即可

✅ 推荐使用 Chrome / Edge / Safari
✅ 安卓 / iOS 手机均可正常使用
📱 使用说明

1. 打开页面，允许摄像头权限
2. 对准商品 → 拖动 / 缩放 → 点击 📸 拍照
3. 填写商品信息 → 点击「✅ 保存」
4. 在「📋 清单」中查看 / 删除记录
5. 完成后点击「📤 导出」下载 Excel + 图片

🧠 技术说明

- 纯原生 HTML / CSS / JavaScript
- 使用 
"getUserMedia" 调用摄像头
- 使用 
"XLSX + JSZip + FileSaver" 导出数据
- 数据存储在 
"localStorage"

⚠️ 注意事项

- 请勿在「无痕模式」下使用（可能导致数据无法保存）
- 清除浏览器缓存会清空所有盘点数据
- 建议定期使用「📥 备份」功能
